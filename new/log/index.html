# 2021.2.4

文件储存解释 (新版)

天翼云盘国内，速度快，部分视频可直接播放，但有封链接的可能。
OneDrive国外，速度快(使用了新CDN)，MP4视频可直接播放，正常不会挂。
Github Release国外，速度玄学，视网络情况而定，直链，90%不会挂。可使用工具加速。
Coding 制品库国内，速度快，直链，75%不会挂，因为要存其它东西，可能会删除旧文件。
Coding Issues国内，速度快，直链，75%不会挂，不知道有多少空间，链接获取较麻烦。
